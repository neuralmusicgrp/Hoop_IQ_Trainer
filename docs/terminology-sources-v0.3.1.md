# Terminology Sources - v0.3.1

This update corrects the original coach homework terminology in Hoop IQ Trainer.

Primary references used:
- The Basketball Action Dictionary PDF
- The Basketball Action Dictionary: Chicago
- NBC Sports Pistol action breakdown
- The Hoops Geek Basketball Plays Library

Key corrections:
- Slash corrected from generic hard cut to pick-and-pop into weak-side DHO.
- Pistol tightened as early-offense sideline action with DHO / keep / ball-screen options.
- Zoom tightened as screen-into-DHO action.
- Spain / Stack tightened as PNR plus back screen for the roller.
- Ram tightened as off-ball screen for the future ball screener.
- Step-Up, Veer, UCLA, Zipper, Brush, and Gortat clarified.
- Bulldog Motion and Euro Ball Screen remain coach/team-specific until confirmed by the coach.

Expanded IQ terms such as Chicago, Miami, Double Drag / 77, Slip, Curl, Fade, Over, Under, Switch, Hedge, Drop, Ice, Blitz, and Weak are planned for a later version.
