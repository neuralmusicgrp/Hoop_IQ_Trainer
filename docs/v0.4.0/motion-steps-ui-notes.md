# Motion Steps UI Notes - v0.4.0

This checkpoint adds the first visible Motion Steps panel to Diagram Study.

What changed:

- Diagram Study now shows a Motion Steps panel when a term has motionSteps data.
- Players can tap Step 1, Step 2, Step 3, etc.
- The court diagram updates from the selected motion step.
- Each step displays a description and coaching checkpoint.
- normalizeTerms now preserves tags, role, and motionSteps metadata.
- Categories are now derived from the default terms so Defensive Coverage and Defensive Concept appear in filters.

This is still step-by-step diagram playback, not full animation.
