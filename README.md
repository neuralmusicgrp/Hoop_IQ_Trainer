# Hoop IQ Trainer

A mobile-friendly basketball terminology trainer for youth players.

Hoop IQ Trainer helps players study coach-assigned basketball terminology through flashcards, quizzes, category training, missed-answer review, progress tracking, and visual half-court diagrams.

## Current Version

v0.3.0 - Visual Basketball IQ

## Features

- Single-file HTML app
- Works offline
- Mobile-friendly design
- Flashcard mode
- Mixed quiz mode
- Category quiz mode
- Missed-term review
- Daily 10 workout
- Progress dashboard
- Coach Edit Mode
- Import/export terminology JSON
- Half-court diagram study
- Name This Action diagram quiz
- Local progress saved with browser localStorage
